The default [LICENSE](./LICENSE) for `magic-cc-codex-worker` is **PolyForm Noncommercial 1.0.0**. It is free to use, modify, and redistribute **for noncommercial purposes**, with attribution preserved.

## When you need a commercial license

You need a separate commercial license if any of these applies to you:

- Your company (for-profit entity) uses this plugin internally on projects that advance commercial goals, even if the plugin itself is not sold.
- You integrate this plugin — or a derivative of it — into a product, service, or SaaS that you offer to customers.
- You redistribute this plugin, or derivative code based on it, as part of a commercial offering.
- You publish public derivative work (a fork with substantial borrowing, or a re-branded plugin clearly derived from this one) beyond noncommercial experimentation.

## When you do **not** need a commercial license

You are already covered by the default license for:

- Personal projects, hobby work, learning, research, and study.
- Open-source personal projects (where you don't derive commercial advantage).
- Use inside nonprofits, charities, public research, government, and educational institutions (per PolyForm Noncommercial).
- Contributions back to this repository (we welcome PRs — see [CONTRIBUTING.md](./CONTRIBUTING.md)).

## What attribution looks like

Whether you use the default license or a commercial license, you must preserve the following notice in any copy or substantial redistribution:

```

Source: https://github.com/wenqingyu/magic-cc-codex-worker
```

If you publish a derivative work or borrow substantial ideas / architecture, please also add a visible reference in your README:

> Based on / inspired by [magic-cc-codex-worker](https://github.com/wenqingyu/magic-cc-codex-worker) by Wenqing Yu.

## How to request a commercial license

Open a GitHub issue on this repo with the label `commercial-license`, or contact the author through the GitHub profile at <https://github.com/wenqingyu>.

Please include:

- Your company / organization name and website
- Intended use (internal tooling, product integration, SaaS, redistribution, etc.)
- Expected scale (team size, user count, revenue tier — as applicable)

Most commercial licenses are straightforward, reasonably priced, and do not require giving up your own IP. Nonprofit or early-stage startup use is often approved at no or nominal cost.

## Why this license

The PolyForm Noncommercial license matches the actual intent of this project: **independent developers and researchers should be able to use, study, modify, and share this freely, while commercial exploitation requires a conversation.** It preserves attribution, keeps the code readable and buildable, and lets us sustain the project while keeping the noncommercial community welcome.

If you're an independent developer, hobbyist, researcher, or nonprofit — just use it and enjoy. No paperwork required.
